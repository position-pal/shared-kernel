interface EventSerializer

Interface for serializing and deserializing User and Group-related events using Avro. Provides methods to convert User, Group, and Group-related event objects (e.g., adding/removing members, creating/deleting groups) to byte arrays and back. Implementations of this interface allow for efficient, binary serialization for transport or storage.
